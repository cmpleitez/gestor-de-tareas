---
description: Buscar error
---

1. Primero verifica cual es la pestaña activa
2. Lee mi reporte de error
3. Busca en el error en la pestaña activa
4. Aisla el problema y lee solo en ese sector, si no encuentras aún la causa del error: pasa a leer los elementos padres y 5. dependientes, para rastrear el error en otros lugares
6. Analiza la lógica consecuencial del error
7. Reportame las lineas donde se origina el error
8. Dame una resolución del error
9. Preguntame si puedes proceder
